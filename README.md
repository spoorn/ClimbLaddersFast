# ClimbLaddersFast

Minecraft mod that changes the speed when going up and down ladders/vines/scaffolding/etc.

## Supported Versions

- Minecraft 26.2
- Fabric and NeoForge
- Java 25+

## Building

Use the checked-in Gradle wrapper with Java 25:

```sh
./gradlew build
```

The release jars are produced by the platform `shadowJar` tasks:

- `fabric/build/libs/climbladdersfast-<version>-fabric.jar`
- `neoforge/build/libs/climbladdersfast-<version>-neoforge.jar`
